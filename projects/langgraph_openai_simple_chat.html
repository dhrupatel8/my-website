Simple Chat App with Memory

  • Tech Stack: LangGraph, OpenAI, Docker, Python, HTML, CSS, javascript
  • Github URL: Project Link

This project is a **memory-enabled conversational app** built with LangGraph, OpenAI, and FastAPI. It lets users interact with an AI model through a simple web interface while maintaining live memory of the conversation. Each session has a unique ID, so the model can track context across multiple turns, and conversations are persisted even after refreshing or restarting the app.

A standout feature of this app is its **live memory visualization**. The right-hand panel in the web UI shows the active conversation memory in real time, and users can archive previous sessions to start fresh without losing history. Archives are stored on disk and can be retrieved later, making it easy to switch between different conversations while preserving context.

The project is fully containerized with Docker and comes with a ready-to-run `docker-compose` setup. It also includes endpoints for graph visualization, allowing developers to inspect the underlying LangGraph workflow. Whether you want to experiment locally, deploy on cloud infrastructure, or extend the app with new tools and retrieval nodes, this starter provides a solid foundation for building advanced, stateful AI applications.
