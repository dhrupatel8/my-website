Sepsis Prediction

  • Tech Stack: Python, SQL, Spark, BigQuery, Pytorch
  • Github URL: Project Link
  • Youtube URL: Youtube Link

Developed Machine Learning model to predict sepsis in patients (MIMIC-III dataset) before 18 hours. Accuracy 99%.

This study presents an ML approach to predict sepsis-onset-time using features like vitals, GCS, and lab results for ICU patients.

This study quantifies the predictive power of each feature-set. This helps with the interpretability of the model, as clinicians can gauge which feature is adding the most value.
